发行说明

3.5.2

优化了段落回退按钮

段落回退按钮现在第一次点击会回到段落开始，两秒之内第二次点击会回到上一个段落。

优化了翻译的显示

之前的翻译有点抢戏，而且和文本区别不明显，现在将翻译的颜色改淡了，字体大小改小了一些。

优化了隐藏文本

现在隐藏文本后，如果点击显示了文本，支持重新隐藏文本了。

其他 bug 修复和体验优化

  1. 优化了播放页面音频详情的显示。
  2. 修复了修改文本时可能出现的问题。
  3. 优化了视频在横屏状态下的布局。
  4. 优化了收藏句子播放器的体验。
  5. 优化了缓存清理逻辑。

3.5.1

优化了播放页面的性能

重构了部分播放页面的代码以优化播放页面的性能，目前测试下来有小幅的提升，不再那么卡了。

首页支持了模块化展示

首页左上角增加了一个编辑按钮，开启后可以调节模块的显示和顺序，导入文件模块移动到右上角，增加了播客媒体库的导入。

优化了收藏的句子的播放体验

现在收藏的句子有了新的独立的播放页面，支持调节速度和重复次数。

增加了编辑文本的功能

因为文本可能出错，很多朋友都想要编辑文本，现在支持编辑错误的文本了。

其他 bug 修复和体验优化

  1. 修复自动翻译的部分 bug，翻译显示不全等。
  2. 间隔重复增加显示下次复习时间。
  3. 优化词汇导出，新增句子字段。
  4. 支持音频导出。
  5. 修复部分文本和音频完全匹配错误问题。
  6. 修复文件下载缓存相关问题。
  7. 修复长文本导出 PDF 的问题。
  8. 导入的文件支持排序。

3.4.1

优化了自动翻译

自动翻译的准确度和体验之前版本一直不太好，这个版本做了大幅优化，希望能够帮助到你。

支持导出文本为 PDF 文档

现在文本支持导出为 PDF 文档，目前排版可能还不够好，欢迎大家的反馈，后续会持续优化。

优化了句子收藏

收藏过的句子现在会在段落顶部显示标识，也增加了取消收藏的操作。

bug 修复和体验优化

会员购买页增加了付费免费对比说明，播放小组件现支持直接点击播放（iOS 17），查词增加了震动反馈配置，统一长按文本操作的逻辑，其他问题修复。

3.4.0

新增了网页翻译

提供了多种网页翻译方式，包括 deepl，微软，百度，搜狗，谷歌，yandex。如果觉得自带的翻译不准确，可以使用这个功能对比下，在段落旁边的三个点按钮菜单中开启。

新增了音频波形控制器

有些用户希望更精确的调整时间，因此新增了这个控制器，目前体验还未做到最佳，有待优化，后续还可以在此基础上拓展更多功能，在播放页面左下角的按钮菜单中开启。

bug 修复和体验优化

文件导入支持了多选，支持隐藏已经播放过的单集，单词页面支持切换英文释义（长按中文释义），增加了保持可见的滚动模式，长按查词模式增加点击单词播放，其他问题修复。

3.3.2

修复了老设备的崩溃问题

十分感谢用户帮我测试，终于修复了这个问题！

新增了常见问题页面

把大家在使用 app 时经常问我的问题做了个汇总，放到“更多”>“关于”>“常见问题”，有什么疑问可以先去看看。

bug 修复和体验优化

优化了音频详情页面的展示，优化了模型选择页面的设计，修复了一个从app发布一直存在到上个版本的 bug。

3.3.1

优化词汇页面

词汇页面进行了翻新，“生词库”和“音频词汇”整合在一起了，同时收藏的句子也进行了类似的整合。去掉了没有什么用的词频统计。

优化了句子收藏

现在无论是随单词收藏的例句，还是手动收藏的句子都不会因为删除音频而导致失去原声，句子还支持了导出音频。

增加了间隔重复

间隔重复是目前公认的比较好的用于记忆的方法，现在查过的生词都会自动加入间隔重复的复习队列中，大家听完了播客记得有时间复习一下单词，不然这里会积累得越来越多。

v3.3.0

bug 修复和体验优化

这个版本主要修复了翻译语言识别错误的问题，又新增一种字体，增加了文字间距的调整，可能稍微优化了文本显示的性能，还有其他小问题的修复。

v3.2.3

新增自动翻译

很多用户需要的全文翻译，现以自动翻译的形式为大家提供，从“更多>音频”中打开自动翻译，应用将在播放中自动为你翻译当前段落。

新增语言学习分类

作为一个语言学习分类，竟然现在才增加语言学习分类？抱歉是我的失误。不过语言学习分类里面有很多多语言的播客，最好在播客详情列表右上角将播客语言改为 auto，这样可以自动识别多语言。

新增两种播放字体

一直在寻找适合阅读的字体，这次给大家提供了两种新的选择，也欢迎推荐你认为不错的字体给我。

v3.2.2

新增大家在看

增加了大家在听的板块，如果不知道听什么，可以看看大家在听的内容，没准会有新发现，并且文本大部分已经为你提前转译完成，无需等待。

增加长按查词

部分用户觉得点击查词容易误触，因此增加了长按查词的配置，如果需要请前往“更多>音频”中开启。

新增英文转译模型

对于英文的转译，现在增加了专门针对英文优化的模型，请前往“更多>音频>转译设置”中下载使用，对准确率可能会有提升。

v3.2.0

新增快捷翻译按钮

很多用户觉得翻译按钮不方便，所以在文本段落后新增了一个快捷翻译按钮，此功能需要在“更多>音频>显示快捷翻译按钮”中开启。

v3.1.3

优化播客语言的识别

某些播客语言可能会识别不准确，现在你可以在播放页面右上角的菜单中为播客重新选择语言。

词汇导出改为 csv

词汇现在导出格式为 csv，可以导入到 anki 软件中进行记忆。也支持单篇播客的词汇导出了。

v3.1.0

云端转译来了

本地转译的准确率和性能一直不太让人满意，于是我决定通过云端转译解决这个问题，现在你经常听的播客会提前在云端静默的为你转译。

支持瑞典语和意大利语

增加了瑞典语和意大利语的基础支持。

v3.0.2

支持导入视频和音频

千呼万唤始出来，很多用户的期待的导入音频和视频的功能来了，需要的快去试试吧。

v3.0.1

支持通过 URL 收藏播客

现在支持通过 URL 来收藏播客啦，某些搜索不到的播客或者私有链接的播客可以通过这种方式添加。

部分播客支持章节跳转

对于提供了时间戳的播客，现在支持章节跳转。如果该播客支持跳转，播放页面右上角会出现章节按钮。

v3.0.0

统一设计语言

统一了整个 app 的设计语言，重构了词汇页面，简化了一些不必要的设计。

增加词频统计

词汇页面增加了“词频统计”板块，统计你听过的播客的文本的所有出现的单词的次数，目前还没有其他衍生功能。

新增时间统计

现在会统计你的收听时间，积少成多，希望可以帮助你了解自己在听播客上花的时间。

新增 App 图标

继续给大家带来几个看起来还不错的图标，去挑挑看有没有你喜欢的吧。

v2.3.0

bug 修复和体验优化

修复了快速播放小组件有时候显示异常的问题，优化了反馈问题页面的设计细节等问题。

v2.2.3

重构首页布局

一直对首页布局不太满意，用户也希望把收藏的播客放到首页，于是就有了这次的改动。去掉了正在播放的板块，增加了收藏的播客、最新单集、收藏的音频板块。

优化了音频状态的显示

对音频增加了正在播放、播放过、未播放三种状态的区分显示。

共享单词和句子

通过共享单词和共享句子的按钮，你可以调用其他应用对文本进行处理，比如调用欧路词典查词等等。

重构了大爆炸页面

去掉了大爆炸的选词页面，替换为系统的长按选择文本模式，现在可以更方便的选词了。

v2.2.2

紧急修复转译 bug

上个版本修改转译器导致了一个严重 bug，十分抱歉，这个版本主要是修复这个问题。

优化查词弹窗设计

在用户的督促下，把查词弹窗的设计优化了，现在看起来更协调。

v2.2.1

支持收藏句子

重构了单词高亮

增加了动画和圆角，在隐藏式菜单未打开的情况下，支持长按选择文字。

新版转译器

我们一直在优化转译器的体验，现在推出新版转译器，对比旧版各有优劣，请前往更多>音频里面打开体验。

v2.2.0

bug 修复和体验优化

修复一些 bug，优化了一些体验，修改了 iPad 上的布局模式。
